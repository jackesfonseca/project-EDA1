# TutorChess
Um projeto voltado a ensinar o usuário a jogar xadrez de forma simples e amigável.

### Instalação
```sh
$ pip install -r requirements.txt
```

### Uso do Programa Principal
```sh
$ cd grupo-1-ai
$ python src\Program.py
```

### Testes
```sh
$ cd grupo-1-ai
$ python src\Program_tests.py -v
```
